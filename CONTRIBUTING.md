# Hướng dẫn đóng góp bài giảng

## Ai có thể đóng góp?
- Giáo viên
- Sinh viên sư phạm
- Nhóm giáo dục độc lập

## Nguyên tắc
- Nội dung đúng chương trình phổ thông
- Không sao chép sách thương mại
- Dùng theme Học Đường Việt
- Tôn trọng bản quyền

## Cách đóng góp
1. Fork repo
2. Tạo thư mục bài giảng mới
3. Viết `metadata.yaml`
4. Viết `slides.tex` và `lesson.tex`
5. Tạo Pull Request

## Ghi nhận
- Tên tác giả hiển thị công khai
- Giữ lịch sử chỉnh sửa
